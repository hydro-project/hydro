Enum HydroDeploy 

Source 
pub enum HydroDeploy {}
Available on crate feature deploy only.
Expand description

Deployment backend that uses hydro_deploy for provisioning and launching.

Automatically used when you call crate::compile::builder::FlowBuilder::deploy and pass in an &mut reference to hydro_deploy::Deployment as the deployment context.
