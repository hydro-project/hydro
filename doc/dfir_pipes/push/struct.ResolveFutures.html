Struct ResolveFutures

pub struct ResolveFutures<Psh, Queue, QueueInner> { /* private fields */ }
Expand description

Push operator that receives futures, queues them, and pushes their resolved outputs downstream.

Queue is expected to be either [futures_util::stream::FuturesOrdered] or [futures_util::stream::FuturesUnordered] (or a mutable reference thereof).
