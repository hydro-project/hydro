Expand description

Accumulator state for reduce.

  • Owned mode (Accum = Option<T>): the first item initializes the accumulator, subsequent items are merged. Takes the value on finalize, emitting 0 or 1 T downstream.
  • Borrowed mode (Accum = &'a mut Option<T>): same accumulation logic, but the option persists across ticks. Emits &'a mut T on finalize (or nothing if no items were received).
