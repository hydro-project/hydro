Struct SlotVec 

Source 
pub struct SlotVec<Tag: ?Sized, Val> { /* private fields */ }
Expand description

A Vec-based SlotMap-esque datastructure without removes.

Analogous to slotmap::SlotMap, but avoids the overhead of tracking removed keys.
