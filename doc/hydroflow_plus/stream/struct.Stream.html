pub struct Stream<T, B, N> { /* private fields */ }
Expand description

An infinite stream of elements of type T.

Type Parameters:

  • T: the type of elements in the stream
  • B: the boundedness of the stream, which is either Bounded or Unbounded
  • N: the type of the node that the stream is materialized on
