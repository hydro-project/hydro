Expand description

Virtual COLT “node” with an API similar to GeneralizedHashTrieNode.

The “current node” at depth d from the root in the Wang/Willsey/Suciu paper sits above an unbalanced trie. We emulate that here via a variadic of nodes at depth d in all the tries in the forest (that are height d or taller.) This is basically a GhtForest of subtries.

source

fn get(self, head: &Self::Head) -> Self::Get

On an Inner node, retrieves the value (child) associated with the given “head” key. returns an Option containing a reference to the value if found, or None if not found. On a Leaf node, returns None.
